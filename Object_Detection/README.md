# Object Detection and Tracking using YOLOv8

## Description
This project performs object detection and multi-object tracking using the YOLOv8 model by Ultralytics.

## Features
- Object Detection
- Object Tracking
- Bounding Boxes
- Tracking IDs
- Video Processing

## Requirements

- ultralytics
- opencv-python

## Run

Install the required libraries:

```bash
pip install ultralytics opencv-python
```

Run:

```bash
python app.py
```
