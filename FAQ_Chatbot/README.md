# FAQ Chatbot

## Description

This project is a simple FAQ chatbot developed using Python. It answers frequently asked questions by comparing user input with stored questions using TF-IDF Vectorization and Cosine Similarity.

## Features

- FAQ-based chatbot
- TF-IDF Vectorization
- Cosine Similarity
- Python

## Requirements

- scikit-learn

## Run

Install dependencies:

```bash
pip install scikit-learn
```

Run:

```bash
python app.py
```
